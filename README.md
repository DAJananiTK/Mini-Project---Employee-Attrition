# Employee Attrition Analysis Project

## 📌 Description
This  project focuses on analyzing employee attrition data to understand the factors affecting employee turnover. 
The project involves data cleaning, transformation, and visualization.

## 📁 Files Included
- raw_data.xlsx - Original employee dataset  
- clean_data.xlsx - Cleaned dataset using Excel and Power Query  
- report.pdf / document.docx - Project documentation  

## 🔧 Work Done
- Data cleaning using Microsoft Excel  
- Data transformation using Power Query  
- Data visualization and report creation  
- Analysis of employee attrition patterns  

## 📊 Tools Used
- Microsoft Excel  
- Power Query  
- Microsoft Word  

## 🎯 Objective
To identify key factors influencing employee attrition and present insights through visual reports
Factors include employee demographics,compensation,training details and satisfaction parameters

## 📈 Outcome
- Cleaned and structured dataset  
- Interactive visualizations  
- Clear insights on employee attrition
